# Version

0.10.1

## 2026-07-16 생산일보 간소화

- 생산일보 입력에서 작업 공정과 생산수량을 제거했습니다.
- 출고완료 명칭을 생산완료로 바꾸고 생산중·생산완료 상태로 단순화했습니다.
- 작업내용을 작업내용 및 기타사항으로 넓혀 기존처럼 자유롭게 기록할 수 있습니다.
- 기존 생산일보는 그대로 조회하며 수정 후에는 간소화된 항목으로 저장합니다.

## 2026-07-16 휴대폰 설치형 업무앱

- `/admin` 업무포털을 아이폰과 안드로이드 홈 화면에 설치할 수 있는 PWA로 확장했습니다.
- 휴대폰 하단에 홈, 현장, 견적, 재고, 전체 메뉴를 고정해 한 손 조작을 빠르게 만들었습니다.
- 현장사진 촬영, 견적·재고 입력, PDF 공유 팝업을 작은 화면과 안전영역에 맞췄습니다.
- 업무 API와 민감자료는 오프라인 캐시에 저장하지 않고 로그인 세션을 계속 거치도록 유지했습니다.

## 2026-07-15 통합 업무 운영

- 홈페이지 관리자 포털에 현장 관리, 생산 관리, 재고 관리, 견적 관리를 하나의 왼쪽 메뉴로 통합했습니다.
- 견적 일정, 생산일보, 제품 재고와 입출고 원장을 Private Blob에 저장합니다.
- 직원별로 네 업무 권한을 나누고, 통합 현황에서 실제 운영 건수를 확인합니다.

## 2026-07-15 직원 계정·권한 관리

- 총책임자가 직원 전화번호를 아이디로 등록하고 업무별 권한을 선택합니다.
- 직원은 최초 등록코드로 본인 비밀번호를 직접 만든 뒤 전화번호와 비밀번호로 로그인합니다.
- 퇴사·휴직 직원은 총책임자가 사용 중지하며 기존 로그인 세션도 다음 서버 요청부터 차단됩니다.
- 비밀번호 분실 시 총책임자가 기존 비밀번호를 무효화하고 새 등록코드를 발급합니다.
- 직원 비밀번호와 등록코드는 Private Blob에 `scrypt` 해시로만 저장합니다.

## 2026-07-15 PC 카카오톡 붙여넣기

- PC에서는 노란색 `카톡에 붙여넣기`를 누르면 작성 문서가 고화질 이미지로 복사됩니다.
- 카카오톡 PC 채팅창에서 `Ctrl+V`를 누르면 별도 파일 메뉴 없이 바로 붙여넣을 수 있습니다.
- 휴대폰에서는 기존처럼 PDF 파일 공유창을 사용합니다.

## 2026-07-15 PDF 저장·카카오톡 공유

- 견적서와 거래명세서를 브라우저 인쇄창이 아닌 실제 PDF 파일로 생성합니다.
- 휴대폰에서는 시스템 공유창을 열어 카카오톡 채팅방으로 PDF를 보낼 수 있습니다.
- PC처럼 파일 공유를 지원하지 않는 환경에서는 PDF를 내려받고 카카오톡 파일 첨부를 안내합니다.
- PDF 생성 과정은 현재 브라우저 안에서 처리되며 작성 문서를 외부 저장소에 업로드하지 않습니다.

## 2026-07-15 일정·현장 업무일지

- 관리자 `일정·할 일`을 월간 달력 기반 `일정·현장 업무일지`로 확장했습니다.
- PC와 휴대폰에서 같은 일정, 작업상태, 현장, 참여자와 작업내용을 확인할 수 있습니다.
- 휴대폰 카메라로 촬영하거나 사진을 선택하면 자동으로 용량을 줄여 비공개 저장소에 등록합니다.
- 저장된 사진은 관리자 로그인 세션을 통과한 경우에만 표시합니다.
- 일정별 `.ics` 파일을 내려받아 휴대폰 기본 캘린더에 추가할 수 있습니다.

## 2026-07-15 웹 견적서·거래명세서

- 관리자 `문서 작성` 화면에서 견적서와 거래명세서를 바로 작성할 수 있습니다.
- 수량과 단가에 따른 공급가액, 부가세, 합계금액과 한글 금액을 자동 계산합니다.
- 작성 내용은 현재 브라우저에만 임시저장하고 `PDF 저장 / 인쇄`로 A4 출력 화면을 엽니다.
- 품목 행 추가·삭제, 문서 종류 전환, 원본 XLS 다운로드를 함께 제공합니다.

## 2026-07-14 관리자 업무 서식

- 견적서와 거래명세서 원본 `.xls` 양식을 관리자 전용으로 등록했습니다.
- 관리자 `업무 서식` 메뉴와 견적 관리 화면에서 내려받을 수 있습니다.
- 원본은 Public GitHub가 아닌 Vercel Private Blob에 보관하고 로그인 세션이 확인된 경우에만 전송합니다.
- 임의 경로 요청은 허용하지 않고 지정된 두 서식만 제공합니다.

## 2026-07-14 관리자 업무포털

- `/admin` 내부 업무포털을 추가했습니다.
- 견적 ERP와 세무·회계 ERP의 연결 상태를 한 화면에서 확인할 수 있습니다.
- 관리자 비밀번호 해시와 8시간 서명 세션을 적용했습니다.
- 실제 ERP가 연결되기 전에는 회사 자료나 가상 수치를 저장하지 않습니다.
