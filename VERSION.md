# Version

0.5.0

## 2026-07-15 웹 견적서·거래명세서

- 관리자 `문서 작성` 화면에서 견적서와 거래명세서를 바로 작성할 수 있습니다.
- 수량과 단가에 따른 공급가액, 부가세, 합계금액과 한글 금액을 자동 계산합니다.
- 작성 내용은 현재 브라우저에만 임시저장하고 `PDF 저장 / 인쇄`로 A4 출력 화면을 엽니다.
- 품목 행 추가·삭제, 문서 종류 전환, 원본 XLS 다운로드를 함께 제공합니다.

## 2026-07-14 관리자 업무 서식

- 견적서와 거래명세서 원본 `.xls` 양식을 관리자 전용으로 등록했습니다.
- 관리자 `업무 서식` 메뉴와 견적 관리 화면에서 내려받을 수 있습니다.
- 원본은 Public GitHub가 아닌 Vercel Private Blob에 보관하고 로그인 세션이 확인된 경우에만 전송합니다.
- 임의 경로 요청은 허용하지 않고 지정된 두 서식만 제공합니다.

## 2026-07-14 관리자 업무포털

- `/admin` 내부 업무포털을 추가했습니다.
- 견적 ERP와 세무·회계 ERP의 연결 상태를 한 화면에서 확인할 수 있습니다.
- 관리자 비밀번호 해시와 8시간 서명 세션을 적용했습니다.
- 실제 ERP가 연결되기 전에는 회사 자료나 가상 수치를 저장하지 않습니다.

## 메모

- 회사소개와 고객지원 카드에 프로젝트 전용 생성 이미지를 적용했습니다.
