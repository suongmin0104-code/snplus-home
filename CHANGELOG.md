# Changelog

## 2026-07-16 - 0.11.0

### Added

- 관리자 포털 안에서 바로 여는 `/admin/tax` 세무·회계 업무대장
- 월별 입금·출금, 거래 구분, 금액, 거래처, 증빙 확인 상태를 기록하는 비공개 장부
- 부가가치세·원천세·법인세·급여·자료 제출 일정과 담당자, 완료 여부 관리
- 증빙 확인이 필요한 거래와 지연 업무를 한 화면에 모아 보는 초보자용 현황판
- 세무·회계 권한을 가진 직원만 사용하는 전용 API와 Private Blob 저장소

### Changed

- 관리자 포털의 세무·회계 메뉴를 외부 ERP 연결 화면에서 내부 업무대장 진입 화면으로 변경
- 개발 미리보기와 운영 주소에서 세무·회계 버튼이 각각 정확한 내부 화면을 열도록 경로 분리
- 세무 저장 기능을 기존 관리자 업무 API에 통합해 Vercel Hobby의 서버 함수 12개 제한 안에서 운영

### Security

- 세무·회계 화면과 API에 `no-store`, `noindex`, 기존 관리자 세션과 직원별 `tax` 권한 적용
- 회사 장부는 공개 저장소나 브라우저 저장소가 아닌 비공개 서버 저장소에만 보관

### Verified

- 월별 입금·출금·차액, 증빙 누락, 미완료·지연 일정 자동 검사 통과
- `npm run test:tax`, `npm run test:operations`, `npm run test:admin-auth`, `npm run lint`, `npm run build` 통과
