# 세무·회계 내부 업무대장 운영 가이드

## 목적

`https://snplus.ai.kr/admin/tax`는 주식회사 에스앤 직원이 입출금과 증빙, 세무 제출 일정을 한곳에서 정리하는 사내 업무대장입니다. 외부 회계 프로그램으로 이동하지 않고 기존 SN 업무포털 로그인과 직원별 권한을 그대로 사용합니다.

## 직원 사용 순서

1. 돈이 들어오거나 나간 날 `입출금 기록`을 누릅니다.
2. 날짜, 입금·출금, 거래 구분, 금액과 거래처를 적습니다.
3. 영수증이나 세금계산서를 확인했으면 `첨부 확인`, 아직이면 `확인 필요`를 선택합니다.
4. 세무사에게 자료를 보낼 날이나 신고 준비 마감일은 `일정 추가`에 등록합니다.
5. 처리가 끝난 일정만 `완료`로 체크합니다.

## 한눈에 보기

- 선택한 달의 입금 합계
- 선택한 달의 출금 합계
- 입금에서 출금을 뺀 단순 차액
- 처리할 세무·제출 업무와 지연 건수
- 증빙 확인이 필요한 거래
- 총책임자가 월별로 입력한 부가가치세·원천세·법인세·기타세금 준비금과 합계

입출금 현황은 직원이 직접 기록한 단순 합계입니다. 회계상 손익이나 납부세액을 자동 계산하는 값이 아닙니다.

## 대표 예상 세금

`대표 예상 세금`은 총책임자 계정에서만 보입니다. 선택한 조회 월마다 아래 금액을 따로 입력합니다.

- 예상 부가가치세
- 예상 원천세
- 법인세 준비금
- 기타 예상 세금
- `세무사 확인 전 예상치` 또는 `세무사 확인 금액` 상태
- 안내 내용, 납부 예정일 또는 준비 계좌 메모

현금 입출금 장부만으로 세금을 자동 산출하지 않습니다. 세무대리인에게 안내받은 금액이나 회사가 내부적으로 준비할 금액을 대표가 직접 기록하며, 화면의 합계는 네 항목을 더한 준비금입니다. 실제 신고·납부액은 반드시 세무대리인 확인 후 확정합니다.

## 권한과 저장

- 총책임자 또는 `세무·회계` 권한을 받은 직원만 사용할 수 있습니다.
- 예상 세금 금액의 조회와 수정은 총책임자 계정으로 제한합니다. 일반 직원은 `세무·회계` 권한이 있어도 이 금액을 받을 수 없습니다.
- 세무·회계 화면과 API는 검색엔진 노출 및 브라우저 캐시를 차단합니다.
- 실제 자료는 Vercel Private Blob의 아래 경로에 저장합니다.

```text
admin-tax/snplus/transactions/
admin-tax/snplus/tasks/
admin-tax/snplus/forecasts/
```

- 저장·수정 기록에는 처리 직원과 시각을 남깁니다.
- 같은 달을 여러 화면에서 열어 둔 경우 먼저 저장된 최신 값을 보호하며, 오래된 화면의 저장 요청은 거절하고 다시 확인하도록 안내합니다.
- 화면을 벗어나거나 계정이 바뀌면 대표 예상 세금은 즉시 가리고 로그인 권한을 다시 확인합니다.
- 공개 홈페이지, GitHub와 브라우저 `localStorage`에는 회사 장부를 저장하지 않습니다.

## 현재 범위

- 수기 입출금 장부
- 대표 전용 월별 예상 세금 준비금
- 증빙 확인 상태
- 세무·제출 일정과 담당자
- 완료·미완료와 지연 업무 확인
- 데스크톱·태블릿·휴대폰 반응형 화면

## 하지 않는 일

- 홈택스 전자신고 또는 세금 납부
- 통장·카드·세금계산서 자동 수집
- 자동 분개, 결산 또는 재무제표 생성
- 입출금 자료만으로 세액 자동 계산
- 세액과 계정과목의 최종 판단

신고 금액, 세무 처리와 계정과목은 반드시 세무대리인 또는 담당 전문가의 검토 후 확정합니다. 자동 수집이나 전자신고가 필요해지면 공식 API 이용 권한, 감사 로그, 데이터베이스 트랜잭션과 추가 보안 요건을 먼저 설계합니다.
