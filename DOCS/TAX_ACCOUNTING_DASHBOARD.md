# 세무·회계 내부 업무대장 운영 가이드

## 목적

`https://snplus.ai.kr/admin/tax`는 주식회사 에스앤 직원이 입출금과 증빙, 세무 제출 일정을 한곳에서 정리하는 사내 업무대장입니다. 외부 회계 프로그램으로 이동하지 않고 기존 SN 업무포털 로그인과 직원별 권한을 그대로 사용합니다.

## 직원 사용 순서

1. 돈이 들어오거나 나간 날 `입출금 기록`을 누릅니다.
2. 날짜, 입금·출금, 거래 구분, 금액과 거래처를 적습니다.
3. 영수증이나 세금계산서를 확인했으면 `첨부 확인`, 아직이면 `확인 필요`를 선택합니다.
4. 세무사에게 자료를 보낼 날이나 신고 준비 마감일은 `일정 추가`에 등록합니다.
5. 처리가 끝난 일정만 `완료`로 체크합니다.

## 한눈에 보기

- 선택한 달의 입금 합계
- 선택한 달의 출금 합계
- 입금에서 출금을 뺀 단순 차액
- 처리할 세무·제출 업무와 지연 건수
- 증빙 확인이 필요한 거래

표시되는 금액은 직원이 직접 기록한 입출금의 단순 합계입니다. 회계상 손익, 부가가치세, 법인세 또는 납부세액을 계산하거나 확정하는 값이 아닙니다.

## 권한과 저장

- 총책임자 또는 `세무·회계` 권한을 받은 직원만 사용할 수 있습니다.
- 세무·회계 화면과 API는 검색엔진 노출 및 브라우저 캐시를 차단합니다.
- 실제 자료는 Vercel Private Blob의 아래 경로에 저장합니다.

```text
admin-tax/snplus/transactions/
admin-tax/snplus/tasks/
```

- 저장·수정 기록에는 처리 직원과 시각을 남깁니다.
- 공개 홈페이지, GitHub와 브라우저 `localStorage`에는 회사 장부를 저장하지 않습니다.

## 현재 범위

- 수기 입출금 장부
- 증빙 확인 상태
- 세무·제출 일정과 담당자
- 완료·미완료와 지연 업무 확인
- 데스크톱·태블릿·휴대폰 반응형 화면

## 하지 않는 일

- 홈택스 전자신고 또는 세금 납부
- 통장·카드·세금계산서 자동 수집
- 자동 분개, 결산 또는 재무제표 생성
- 세액 계산과 계정과목의 최종 판단

신고 금액, 세무 처리와 계정과목은 반드시 세무대리인 또는 담당 전문가의 검토 후 확정합니다. 자동 수집이나 전자신고가 필요해지면 공식 API 이용 권한, 감사 로그, 데이터베이스 트랜잭션과 추가 보안 요건을 먼저 설계합니다.
