# SN 업무포털 휴대폰 앱 가이드

## 주소

```text
https://snplus.ai.kr/admin
```

PC와 휴대폰은 같은 계정과 같은 업무 데이터를 사용합니다. 휴대폰 앱은 별도의 복사본이 아니라 현재 관리자 포털을 설치형 화면으로 여는 PWA입니다.

## 안드로이드 설치

1. Chrome에서 관리자 주소를 엽니다.
2. 카카오톡, 네이버, 문자 앱 안에서 열었다면 상단 휴대폰 아이콘의 `설치 안내`를 누르고 `Chrome에서 열기`를 선택합니다.
3. 로그인 후 상단 휴대폰 아이콘 또는 `앱 설치`를 누릅니다.
4. 설치 확인을 누르면 홈 화면에 `SN 업무` 아이콘이 생성됩니다.
5. 설치 버튼이 보이지 않으면 Chrome 메뉴에서 `앱 설치` 또는 `홈 화면에 추가`를 선택합니다.

`웹앱을 설치할 수 없음`이 표시되면 카카오톡·네이버 등 앱 내부 브라우저가 아닌 Chrome 주소창에서 `https://snplus.ai.kr/admin`을 직접 여는지 먼저 확인합니다.

## 아이폰 설치

1. Safari에서 관리자 주소를 엽니다.
2. Safari 아래쪽의 공유 버튼을 누릅니다.
3. `홈 화면에 추가`를 선택합니다.
4. 이름이 `SN 업무포털`인지 확인하고 추가합니다.

아이폰의 Chrome에서는 설치 메뉴가 제한될 수 있으므로 Safari를 사용합니다.

## 휴대폰 메뉴

- `홈`: 현장, 생산, 재고, 견적 요약
- `현장`: 일정·업무일지와 현장사진 촬영
- `견적`: 날짜별 견적 일정과 담당 정보
- `재고`: 제품사진, 현재 수량, 입고·출고 원장
- `전체`: 생산, 세무, 거래처, 문서, 서식, 직원권한 등 모든 메뉴

전체 메뉴가 열린 상태에서 오른쪽 회색 영역을 누르면 닫힙니다.

## 현장사진과 문서 공유

- 현장일보의 `촬영·사진 선택`은 휴대폰 후면 카메라를 우선 사용합니다.
- 사진은 업로드 전에 자동으로 줄여 데이터 사용량을 낮춥니다.
- 견적서와 거래명세서는 `PDF 저장`으로 내려받거나 휴대폰 공유창을 열어 카카오톡으로 보낼 수 있습니다.

## 보안 원칙

- 앱 아이콘과 정적 화면만 휴대폰 캐시에 저장합니다.
- 로그인 응답, 업무 API, 현장사진 원본, 견적·재고 데이터는 오프라인 캐시에 저장하지 않습니다.
- 인터넷이 끊기면 저장된 업무내용 대신 연결 안내 화면을 표시합니다.
- 공용 휴대폰에는 설치하지 말고, 분실 시 해당 직원 계정을 즉시 `사용 중지`합니다.

## 앱스토어 버전

현재 PWA는 내부 업무에 바로 사용하기 가장 빠른 형태이며 아직 Google Play에 등록되어 있지 않습니다. Play 스토어 배포가 필요하면 현재 PWA를 Android Trusted Web Activity로 포장하고 앱 서명, Digital Asset Links, Play Console 심사를 추가합니다. 추후 푸시알림, 생체인증, 백그라운드 사진 업로드가 필요해지면 같은 패키지에서 확장합니다.
